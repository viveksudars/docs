### Insert the Mesh SDK script tag

Add the following to the footer of every page you'd like Mesh to appear. 
Make sure to replace your SDK key at the end of the script with the one in the Settings of your Mesh dashboard.

This approach is best suited for marketing sites. 
If you use Google Tag Manager you can add this script directly in the Tag Manager console as a new tag.

There are two options for tracking, cookie-based or fingerprint-based. 
Cookie-based tracking can offer higher success in recognizing visitors across sessions, but if users opt out of cookies they will not be tracked across sessions. 
Fingerprint-based tracking uses device markers to identify if a user has previously visited the site - which will work regardless of cookie preference but has a small chance of not recognizing a returning visitor.

<CodeGroup>

```js Cookie-based Tracking
<script>
(function (m,e,s,h,a,i,c) {
  m[a] = m[a] || function () {
    (m[a].q = m[a].q || []).push(arguments);
  };
  var o = document.createElement(s);
  o.type = 'text/javascript';
  o.id = 'mesh-analytics-sdk';
  o.async = true;
  o.src = h;
  o.setAttribute("data-mesh-sdk", i);
  o.setAttribute("data-mesh-sdk-attributes", JSON.stringify(c));
  var x = document.getElementsByTagName(s)[0];
  x.parentNode.insertBefore(o, x);
})(window,document,'script',
   'https://cdn.jsdelivr.net/npm/@mesh-interactive/mesh-sdk@latest/dist/umd/index.js','mesh',
   'YOUR SDK KEY HERE',
   {useFingerprint: false, track:{session: true, forms: true}});
</script>
```
